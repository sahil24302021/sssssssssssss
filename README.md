## What is Nivanth?

**Nivanth** is an all-in-one event management platform designed for hackathons, tech conferences, lectures, and community-run events. It handles everything — from creating a beautiful event page and collecting registrations to scanning QR codes at the door, managing staff, and scoring project submissions.

Built by **NIVANTH Labs**, a community running tech & design events across **12 cities in India**, Nivanth was born out of the team's own frustration with fragmented event tools. Instead of juggling Google Forms, Excel sheets, WhatsApp groups, and manual check-ins — Nivanth brings it all under one roof.

---

## Why Nivanth?

Most event tools solve **one** thing. Nivanth solves **everything**:

| Pain Point | Old Way | Nivanth Way |
|---|---|---|
| Event pages | Build from scratch or use clunky templates | Beautiful, customizable event pages with live countdown, schedule, speakers, and FAQs — ready in minutes |
| Registration | Google Forms + manual tracking | Built-in registration with custom fields, approval workflows, team/solo support, and capacity tracking |
| Attendance | Paper sign-in or manual Excel | QR-code based check-in with live scanner, real-time stats, and staff portal |
| Team management | WhatsApp chaos | Native team creation with member roles, auto-fill leader info, and size controls |
| Judging | Offline scoring, lost sheets | Project submission portal with scoring system — judges see submissions in one place |
| Staff coordination | No system | Role-based staff portal — scanner, manager, coordinator, volunteer — each with the right tools |

## Core Features

### ~Event Page Builder
Create stunning event pages without writing a single line of code. Customize everything — cover images, color themes (12+ presets including dark mode), fonts (Syne, Playfair Display, Outfit, Space Grotesk), patterns, and animations. Your event page includes:
- **Live countdown timer** to build excitement
- **Schedule timeline** with session details
- **Speaker profiles** with avatars and roles
- **FAQ section** with expandable answers
- **Social links** (Instagram, Twitter/X, LinkedIn, Website)
- **Capacity indicator** with registration progress bar

### ~ Smart Registration System
- Custom form builder with 11 field types (text, email, phone, URL, dropdown, multi-select, file upload, date, and more)
- **Solo or team registration** — configure min/max team size, member fields, and leader auto-fill
- **Manual approval mode** — review applications before accepting
- Private events with invite-only access
- Real-time registration count and capacity tracking

### ~ QR Attendance & Staff Portal
- **Live QR scanner** — camera-based scanning with instant check-in feedback
- **Manual UUID input** — fallback for damaged QR codes
- **Real-time dashboard** — total registered, checked in, remaining — all updating live
- **Role-based staff access**:
  - `Scanner` — QR scanning + manual check-in
  - `Manager` — full attendance overview + search + filters
  - `Coordinator` — event coordination view
  - `Volunteer` — basic check-in access

### ~ Project Submission & Judging
Participants can submit their projects directly through Nivanth:
- Team name, track/category, and problem statement
- Presentation/slides URL (Google Slides, Canva, etc.)
- GitHub and demo links
- Judges see all submissions in a clean, scorable interface
- Submissions lock after scoring — no tampering

### ~ Event Dashboard
Full organizer dashboard for managing:
- Multi-event support — run several events from one account
- Registration data with search and filters
- Attendance analytics (present vs. absent, % completion)
- Staff management with unique staff codes
- Real-time WebSocket updates across all views

## How It's Different

**Nivanth isn't a generic event platform.** It's purpose-built for the types of events that tech communities actually run:

- **Hackathon-native** — Team registration, project submissions, judging, and scoring are first-class features, not afterthoughts.
- **Opinionated design** — Every event page looks premium by default. 12 curated color themes, professional typography, and grid-pattern backgrounds mean your event looks like it has a design team behind it.
- **Staff-first thinking** — Most platforms forget that events need on-ground staff. Nivanth gives scanners, managers, coordinators, and volunteers their own tailored views.
- **Real-time everything** — Supabase-powered real-time sync means attendance counts, registration numbers, and check-in feeds update instantly across every screen.
- **No bloat** — No ticket selling, no seating charts, no wedding features. Just the tools you need for hackathons, conferences, and community events.
